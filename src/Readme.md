# CPU Scheduling Simulator - Team 8

A Python-based simulator for various CPU scheduling algorithms, including FCFS, SJF, RR, and MLFQ. This tool calculates performance metrics and generates Gantt charts for visualization.

## Algorithms Implemented
- **FCFS**: First-Come, First-Served
- **SJF**: Shortest Job First
- **RR**: Round Robin
- **MLFQ**: Multilevel Feedback Queue (Queue 1: Q=2, Queue 2: Q=4, Queue 3: FCFS)

   cd OS-CPU-Scheduling-Team8

2. `pip install matplotlib`

## How to run
1. Run the main script from your terminal:
```
python main.py
```
2. Input Process Data:
Follow the on-screen prompts to enter: **Number of processes**, **Arrival time for each process**, and **Burst time for each process**

3. Select algorithm:
When prompted for the scheduling algorithm, select the algorithm you wanted.
